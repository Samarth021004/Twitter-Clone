Full Stack Twitter Clone 

.env file setup 

MONGO_URI= 
PORT=
JWT_SECRET=
NODE_ENV=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=

Clone the repository , npm i in the terminal for both frontend and backend folders followed by npm run dev to start the servers
